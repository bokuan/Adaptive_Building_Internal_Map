Empowering your property by
ABMap

ABMap is a online building floor map system that dynamically displays and switches between interior building floor plans using a multi-level view system, based on user selection. It integrates with scheduling data to dynamically display real-time information about activities and events in rooms, and allows user to search the location of specific events within the schedule.
